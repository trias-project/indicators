This document follows the modelling pipeline to produce a list of introduced species ranked by emerging status.

The emerging status (column em_status) score is encoded as follows:

  • 3: emerging
  • 2: potentially emerging
  • 1: unclear
  • 0: not emerging

3 Preprocessing

Merge results from GAM and decision rules:

em_list <- append(em_gam_list, em_decision_rules_list)

Spread columns for ranking:

# make a data.frame binding all dfs by row
em_df <-
  dplyr::bind_rows(em_list) %>%
  tidylog::select(-c(dplyr::starts_with("growth"), dplyr::starts_with("dr")))

# spread by model
em_df <-
  em_df %>%
  tidylog::pivot_wider(
    names_from = model,
    id_cols = c(taxonKey, year, indicator),
    values_from = em_status
  )

# assign unique emerging score (if GAM is NA, choose decision rules)
em_df <-
  em_df %>%
  tidylog::mutate(em = ifelse(is.na(GAM), decision_rules, GAM),
         method = ifelse(is.na(GAM), "decision_rules", "GAM"))

# spread by years
em_df <-
  em_df %>%
  tidylog::pivot_wider(
    names_from = year,
    values_from = em,
    id_cols = c(taxonKey, indicator),
    names_prefix = paste0("year_")
  )

# spread by indicator
em_df <-
  em_df %>%
  tidylog::pivot_wider(
    names_from = indicator,
    id_cols = taxonKey,
    values_from = starts_with("year"),
    names_prefix = "em_status_"
  )

We also add scientific names:

em_df <-
  em_df %>%
  tidylog::left_join(taxa_names, by = "taxonKey")

To rank taxa equally placed, we use the average of the minimal guaranteed growth of number of observations in all Belgium over all the evaluation years.

# add mean growth to em_df
em_df <-
  em_df %>%
  tidylog::left_join(em_gam_list$gam_observations_be %>%
    tidylog::group_by(taxonKey) %>%
    tidylog::summarize(mean_growth = mean(growth, na.rm = TRUE))) %>%
  tidylog::relocate(.data$mean_growth, .before = .data$canonicalName)

4 Ranking by emerging status

In this section we create the ranking of introduced species based on the emerging status.

4.1 Hierarchical ranking

The ranking is based on the highest emerging status. The following priority rules are applied, in order of importance:

  1. the more recent, the higher priority is
  2. emerging statuses in protected areas are more important than the ones defined over entire Belgium
  3. emerging statuses from occupancy indicators have priority above the ones from occurrence indicator
  4. the higher average minimal guaranteed growth (#occs/year), the higher priority is
ranking <-
  em_df %>%
  dplyr::arrange(
    desc(year_2022_em_status_occupancy_pa),
    desc(year_2022_em_status_observations_pa),
    desc(year_2022_em_status_occupancy_BE),
    desc(year_2022_em_status_observations_BE),
    desc(year_2021_em_status_occupancy_pa),
    desc(year_2021_em_status_observations_pa),
    desc(year_2021_em_status_occupancy_BE),
    desc(year_2021_em_status_observations_BE),
    desc(year_2020_em_status_occupancy_pa),
    desc(year_2020_em_status_observations_pa),
    desc(year_2020_em_status_occupancy_BE),
    desc(year_2020_em_status_observations_BE),
    desc(mean_growth)
  ) %>%
  tidylog::select(
    taxonKey, canonicalName, kingdom, class,
    year_2022_em_status_occupancy_pa,
    year_2022_em_status_observations_pa,
    year_2022_em_status_occupancy_BE,
    year_2022_em_status_observations_BE,
    year_2021_em_status_occupancy_pa,
    year_2021_em_status_observations_pa,
    year_2021_em_status_occupancy_BE,
    year_2021_em_status_observations_BE,
    year_2020_em_status_occupancy_pa,
    year_2020_em_status_observations_pa,
    year_2020_em_status_occupancy_BE,
    year_2020_em_status_observations_BE,
    mean_growth,
    dplyr::everything()
  )
ranking

4.2 Point strategy

We can also rank using a point strategy reflecting the same ranking strategy as before. We apply gain factors to all emerging status scores using the number of observations in 2020 in all Belgium as reference (gain factor = 1). The following gain factors are applied:

indicator year region gain factor
occupancy 2022 protected areas 3
observations 2022 protected areas 2.5
occupancy 2022 Belgium 2.5
observations 2022 Belgium 2
occupancy 2021 protected areas 2.5
observations 2021 protected areas 2
occupancy 2021 Belgium 2
observations 2021 Belgium 1.5
occupancy 2020 protected areas 2
observations 2020 protected areas 1.5
occupancy 2020 Belgium 1.5
observations 2020 Belgium 1 
ranking_pts <-
  em_df %>%
  tidylog::mutate(em_pts = year_2022_em_status_occupancy_pa * 3 +
    year_2022_em_status_observations_pa * 2.5 +
    year_2022_em_status_occupancy_BE * 2.5 +
    year_2022_em_status_observations_BE * 2 +
    year_2021_em_status_occupancy_pa * 2.5 +
    year_2021_em_status_observations_pa * 2 +
    year_2021_em_status_occupancy_BE * 2 +
    year_2021_em_status_observations_BE * 1.5 +
    year_2020_em_status_occupancy_pa * 2 +
    year_2020_em_status_observations_pa * 1.5 +
    year_2020_em_status_occupancy_BE * 1.5 +
    year_2020_em_status_observations_BE) %>%
  dplyr::arrange(
    desc(em_pts),
    desc(mean_growth)
  ) %>%
  tidylog::select(
    taxonKey, canonicalName, kingdom, class,
    em_pts, mean_growth,
    year_2022_em_status_occupancy_pa,
    year_2022_em_status_observations_pa,
    year_2022_em_status_occupancy_BE,
    year_2022_em_status_observations_BE,
    year_2021_em_status_occupancy_pa,
    year_2021_em_status_observations_pa,
    year_2021_em_status_occupancy_BE,
    year_2021_em_status_observations_BE,
    year_2020_em_status_occupancy_pa,
    year_2020_em_status_observations_pa,
    year_2020_em_status_occupancy_BE,
    year_2020_em_status_observations_BE,
    dplyr::everything()
  )
ranking_pts
