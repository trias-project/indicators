Taxonomic distribution emerging species

Damiano Oldoni

Tim Adriaens

2022-09-12

This document follows the ranking pipeline to produce summary tables and graphs about the taxonomic distribution of the species defined as emerging (emerging status = 3).

1 Setup

Load libraries:

library(tidyverse) # To do data science
library(tidylog) # To provide feedback on dplyr functions
library(here) # To find files

2 Get data

Read the tab separated files containing the emerging status output:

em_df <- read_tsv(here::here(
  "data",
  "output", 
  "ranking_emerging_status_hierarchical_strategy.tsv"),
  na = ""
)

Preview:

head(em_df)

3 Get number of emerging species per class, year, location and variable

First, reshape the data:

emerging_df <-
  em_df %>%
  pivot_longer(starts_with("year_"), 
               names_to = "variable",
               values_to = "emerging_status")

Preview:

head(emerging_df)

We calculate the number of emerging species (emerging_status = 3) at class level for each combination of:

  • year
  • variable: occupancy or number of observations
  • area of interest: Belgium or protectred areas
emerging_df <- 
  emerging_df %>%
  group_by(.data$class, 
           .data$kingdom,
           .data$variable) %>%
  summarise(n_emerging = length(.data$emerging_status[.data$emerging_status == 3]))

Add year, variable and area of interest as columns:

emerging_df <- 
  emerging_df %>%
  mutate(
    year = str_extract(.data$variable, pattern = "\\d+"),
    area = str_extract(.data$variable, pattern = "BE$|pa$"),
    variable = str_extract(.data$variable,
                           pattern = "occupancy|observations")) %>%
  mutate(area = recode(.data$area, BE = "Belgium", pa = "Natura2000")) %>%
  relocate(n_emerging , .after = last_col()) %>%
  ungroup() %>%
  arrange(.data$year, desc(.data$n_emerging))

Preview:

head(emerging_df, 20)

Save this output file:

write_tsv(

4 Summaries and graphs

Kingdoms with emerging species:

kingdoms <- emerging_df %>% distinct(.data$kingdom)
kingdoms

Classes with emerging species:

classes <- emerging_df %>% distinct(.data$class, .data$kingdom)
classes

Distribution of classes at kingdom levels:

classes %>%
  group_by(.data$kingdom) %>%
  count() %>%
  arrange(desc(.data$n))

4.1 Occupancy

4.1.1 Belgium

emerging_df_BE_occupancy <-
  emerging_df %>%
  filter(.data$area == "Belgium" & .data$variable == "occupancy")
emerging_df_BE_occupancy
emerging_plots_BE_occupancy <- map(kingdoms$kingdom, function(x) {
  data <- 
    emerging_df_BE_occupancy %>%
    filter(.data$kingdom == x & .data$n_emerging > 0)
  ggplot(data) +
    geom_col(aes(x = class, y = n_emerging, fill = year), position = "dodge") +
    theme(
    axis.text.x = element_text(
      angle = 315,
      hjust = 0,
      vjust = 1
      )
    )
})
