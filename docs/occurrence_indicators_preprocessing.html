Occurrence indicators: pre-processing

Damiano Oldoni

Toon Van Daele

2019-11-14

This document describes the pre-processing of occurrence data of alien species. The outputs of this pipeline will be used as input for modelling.

1 Setup

Load libraries:

library(tidyverse)      # To do data science
library(tidylog)        # To provide feedback on dplyr functions
library(progress)       # To add progress bars
library(here)           # To find files
library(lubridate)      # To work with dates
library(rgbif)          # To get taxa from publisehd unified checklist

2 Get data

We get occurrence data as obtained in trias-project/occ-processing repository.

2.1 Alien species

We load occurrence data related to alien species in Belgium:

df <- read_tsv(
  file = "https://raw.githubusercontent.com/trias-project/occ-processing/master/data/processed/cube_belgium.tsv",
  col_types = cols(year = col_double(),
                   eea_cell_code = col_character(),
                   taxonKey = col_double(),
                   n = col_double(),
                   min_coord_uncertainty = col_double()),
  na = ""
)

We remove columns we will never use:

df <- 
  df %>%
  select(-min_coord_uncertainty)

and rename column n to obs:

df <- 
  df %>%
  rename(obs = n)

3 Pre-processing

3.1 Add informative columns

For better interpretation of the data, we retrieve canonical names of the species whose data we are going to analyze. We also retrieve the class of each alien species to link occurrence data with the related baseline data:

taxon_key <- 
  df %>% 
  distinct(taxonKey) %>%
  pull()
pb <- progress_bar$new(total = length(taxon_key))
spec_names <- map_df(
  taxon_key, 
  function(k){
    pb$tick()
    name_usage(key = k, return = "data")
  }) %>%
  select(taxonKey = key, canonicalName, kingdomKey, classKey)

We also retrieve kingdom and class:

class_key <- 
  spec_names %>% 
  distinct(classKey) %>%
  filter(!is.na(classKey)) %>%
  pull()
pb <- progress_bar$new(total = length(class_key))
kingdom_class <- map_df(
  class_key,
  function(x){
    pb$tick()
    name_usage(key = x, return = "data")
  }) %>%
  select(classKey, class, kingdomKey, kingdom)

and we add them:

spec_names <-
  spec_names %>%
  left_join(kingdom_class, by = c("kingdomKey", "classKey"))

So, these are the covered alien species:

spec_names

Taxa not assigned to any class:

spec_names %>%
  filter(is.na(classKey))

3.2 Extract x-y coordinates from cellcodes

Extract coordinates from column eea_cell_code. For example, the coordinates of cellcode 1kmE3895N3116 are x = 3895 and y = 3116:

df_xy <- 
  df %>% 
  distinct(eea_cell_code) %>%
  bind_cols(
    tibble(
      x = unlist(str_extract_all(unique(df$eea_cell_code), 
                                 pattern = "(?<=E)\\d+")),
      y = unlist(str_extract_all(unique(df$eea_cell_code), 
                                 pattern = "(?<=N)\\d+"))) %>%
      mutate_all(as.integer)
)

3.3 Apply temporal and spatial restraints

Select observations from 1950, start of state of invasion researches

first_year <- 1950

Some occurrence data can be published with erroneous coordinates (see #trias-project/occ-processing#13) or can be assigned to a grid cell out of Belgium. We remove these records:

be_cellcodes <- df_prot_areas$CELLCODE

Apply restraints defined above to both occurrence data:

df <- 
  df %>%
  left_join(df_xy, by = "eea_cell_code") %>%
  filter(year > first_year) %>%
  filter(eea_cell_code %in% be_cellcodes)

and baseline data:

df_bl <- 
  df_bl %>%
  left_join(df_bl_xy, by = "eea_cell_code") %>%
  filter(year > first_year) %>%
  filter(eea_cell_code %in% be_cellcodes)

3.5 Calculate number of observations of native species

Calculate the total number of alien species observations per eea_cell_code, year and class:

df_alien_species <- 
  df %>%
  left_join(spec_names, by = "taxonKey") %>%
  group_by(classKey, eea_cell_code, year) %>%
  summarize(n_alien = sum(obs)) %>%
  ungroup()
df_alien_species

Substract the alien species observations from the baseline data to get number of observations of native species:

df_bl <- 
  df_bl %>%
  left_join(df_alien_species, by = c("classKey", "eea_cell_code", "year")) %>%
  replace_na(list(n_alien = 0)) %>%
  mutate(native_obs = cobs - n_alien) %>%
  mutate(native_obs = if_else(native_obs < 0, 0, native_obs)) %>%
  select(year, eea_cell_code, classKey, native_obs, natura2000)

3.6 Create time series

For each species, define cells with at least one observation:

df_cc <- 
  df %>%
  group_by(taxonKey) %>%
  distinct(eea_cell_code) %>%
  ungroup()

For each species, identify the first year with an observation:

df_begin_year <- 
  df %>%
  group_by(taxonKey) %>%
  summarize(begin_year = min(year))

For each species, combine begin_year and unique eea_cell_code as found above:

df_cc <- 
  df_cc %>%
  left_join(df_begin_year, by = "taxonKey") %>%
  select(taxonKey, begin_year, eea_cell_code)

Preview:

df_cc %>% head()

For each cell (eea_cell_code) and species (taxonKey) we can now create a time series. Notice that at this step we create only time series slots: occurrence data are added at next stage.

make_time_series <- function(eea_cell_code, taxonKey, begin_year, last_year ) {
  expand_grid(eea_cell_code = eea_cell_code,
              taxonKey = taxonKey,
              year = seq(from = begin_year, to = last_year))

}

# create timeseries slots
df_ts <- pmap_dfr(df_cc, .f = make_time_series, last_year = year(Sys.Date()))

## Add data

# add occurrence data
df_ts <- 
  df_ts %>%
  left_join(df %>% select(taxonKey, year, eea_cell_code, obs), 
            by = c("taxonKey", "year", "eea_cell_code"))

# add membership to protected areas
df_ts <- 
  df_ts %>%
  left_join(df_prot_areas %>% select(CELLCODE, natura2000),
            by = c("eea_cell_code" = "CELLCODE"))
# add classKey
df_ts <- 
  df_ts %>%
  left_join(spec_names %>% 
              select(taxonKey, classKey), 
            by = "taxonKey")

# add baseline data (at class level)
df_ts <- 
  df_ts %>%
  left_join(df_bl %>%
              select(year, eea_cell_code, classKey, native_obs),
            by = c("year", "eea_cell_code", "classKey"))

# replace NAs with 0
df_ts <-
  df_ts %>%
  replace_na(list(native_obs = 0, obs = 0))

Preview:

df_ts %>% head(n = 30)

Add column for presence (1) or absence (0)

df_ts <- 
  df_ts %>%
  mutate(pa_native_obs = ifelse(native_obs > 0, 1, 0),
         pa_obs = ifelse(obs > 0, 1, 0))

Arrange order columns:

df_ts <-
  df_ts %>%
  select(taxonKey, 
         year, 
         eea_cell_code, 
         obs, 
         pa_obs, 
         native_obs, 
         pa_native_obs,
         classKey,
         natura2000)
